## Run Locally

Requirements: GNU Make, node.js, npm

Install client dependencies and run locally
```bash
  $ cd tiles
  $ npm install
  $ npm run dev
```

Build server from source and run on port 18080

```bash
  $ cd server
  $ make
  $ ./bin/server
```
